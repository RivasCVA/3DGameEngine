# 3D Game Engine
A 3D Game Engine built using Java, Lightweight Java GL, and OpenGL. Learns and followed the YouTube tutorial series from [ThinMatrix](https://www.youtube.com/playlist?list=PLRIWtICgwaX0u7Rf9zkZhLoLuZVfUksDP). I learned a lot about graphics libraries and about all of the different complex components that go into making a game engine.
